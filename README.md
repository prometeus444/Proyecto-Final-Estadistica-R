# Proyecto-Final-Estadistica-R
Como parte de la asignatura de Programación y Estadística con R del Máster en Bioinformática y Biología Computacional (UAM), hemos desarrollado este análisis sobre el "Procedimiento Perverso".

A través de simulaciones en R, estudiamos cómo la práctica de elegir un test estadístico (T-test vs. Wilcoxon) basándose en una prueba previa de normalidad afecta negativamente a la fiabilidad de los resultados. El proyecto compara este método condicional frente a estrategias incondicionales, cuantificando las alteraciones en el Error Tipo I y la Potencia Estadística.
